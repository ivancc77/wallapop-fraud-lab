# Wallapop Poller

Este script descarga anuncios de la API de Wallapop, calcula un score de riesgo y guarda los resultados en un JSON diario.

## Instalación
pip install -r requirements.txt

## Ejecución Manual
python3 poller.py

## Salida
Genera/Actualiza el archivo: ../ingestion/wallapop_master.json
